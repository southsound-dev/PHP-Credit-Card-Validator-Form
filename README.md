
# PHP Credit Card Validator from made in PHP.
PHP Credit Card Validator from made in PHP.
For testing purposes you can use the following Examples

Mastercard
----------
Pattern: "/5[1-5][0-9]{14}/"
Examples:
5111111111111111
5522111111111111

Visa
----------
Pattern: "/4[0-9]{12}([0-9]{3})?([0-9]{3})?/"
Examples:
4004571528446170
4500040443327765
